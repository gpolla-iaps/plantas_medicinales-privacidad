Política de Privacidad

La aplicación "Propiedades de las Plantas" es una aplicación informativa sobre plantas medicinales y sus propiedades.

Recopilación de datos

Esta aplicación no recopila datos personales de los usuarios.

Uso de la información

La aplicación se utiliza únicamente para consultar información sobre plantas medicinales.

Permisos

La aplicación no solicita permisos sensibles ni accede a información personal del dispositivo.
